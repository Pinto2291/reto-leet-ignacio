# Reto del diplomado de EdTecnica

*****

## Reto Leet

El reto consiste en crear una pagina que usando **javascript**, **HTML** y **CSS** que permita convertir texto en lenguaje de hacker o leet

1. Se debe usar HTML para la pagina principla
2. Usar CSS para cambiar la apariencia de la pagina
3. Usar JavaScript para la funcionalidad de la pagina
4. Crear un respositorio y subir el resultado a la plataforma de EdTecnica

El lenguaje LEET es cuando se cambian las letras normales de un texto por su equivalente en el lenguaje LEET, ejemplo:

            a: "4",
            b: "8",
            e: "3",
            g: "6",
            l: "1",
            o: "0",
            s: "5",
            t: "7",
            z: "2",
